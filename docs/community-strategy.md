# Community strategy

This repo is meant to be seen and reused within the AI-for-Science / autonomous-science
community — not to sit private. The plan is deliberately small and durable.

## Home communities

- **LLM Hackathon for Applications in Materials Science & Chemistry** — the primary home
  turf (materials + chemistry + agents). Active Slack, showcase, and summary papers
  between editions.
- **ML for Microscopy Hackathon (Kalinin group)** — digital-twin + agent + instrument
  control; a strong model for autonomous-instrumentation work.
- **Materials Project ecosystem** — a natural first database/MCP connector target.
- **Hugging Face** — later home for any scientific-agent evaluation benchmark or datasets.

## First contributions (where this project is uniquely positioned)

1. **Scientific agent evaluation** — replayable acceptance contracts for scientific
   reasoning. Rare in the community; central here.
2. **MCP connectors** for science tools (Materials Project, PubChem, Crossref, arXiv).
3. **Demos + architecture docs** — clear, honest, reviewable workflows.

## Participation order

`observer → builder → contributor → collaborator`. Build the artifact first; contribute
evaluation, connectors, and docs; earn a voice by being useful. No overclaiming of
endorsement or affiliation.
